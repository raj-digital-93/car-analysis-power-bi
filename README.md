## Project Overview
This project analyzes car market data to understand pricing, mileage, and engine trends across brands, models, fuel types, and years using Power BI.

## Dataset
The dataset includes:
- Car ID
- Brand
- Model
- Year
- Engine Size
- Fuel Type
- Transmission
- Mileage
- Condition
- Price

## Tools Used
- Power BI Desktop
- Power Query (Data Cleaning)

## Dashboard Features
- Average price analysis by brand
- Top 5 models by average mileage
- Comparison of Mileage, Price, Engine Size by fuel type
- Year-wise price range analysis
- Interactive slicers for condition, fuel type, and transmission

## Key Insights
- BMW has the highest average vehicle price, indicating premium positioning
- Ford Fiesta model ranks highest in average mileage, followed by Model X and Q5
- Hybrid cars show lower average prices as well as offer higher mileage
- Automatic transmission vehicles got slight average price more than Manual transmission but better mileage as compared to manual
- Vehicle prices vary significantly by year
